Machine Learning Chat Bot

An AI-driven chatbot powered by a custom PyTorch model that interprets and responds to user input with Rasa.

What I Learned:

  • Rasa Framework: Gained expertise in using Rasa to create conversational AI applications, focusing on its NLU (Natural Language Understanding) and Core (dialogue management) components.
  • Intent Classification: Learned how to train a custom intent classifier using PyTorch to categorize user input into predefined intents, leveraging neural networks.
  • PyTorch Neural Networks: Implemented a custom neural network model using PyTorch, including defining layers, activation functions (e.g., ReLU), and loss functions (e.g., CrossEntropyLoss) for supervised learning.
  • Training and Optimization: Utilized optimization techniques (e.g., Adam optimizer) and backpropagation to minimize the loss function during the model's training phase, improving the accuracy of the intent classifier.
  • Data Preprocessing: Applied text preprocessing techniques such as tokenization, normalization (e.g., converting to lowercase), and padding to convert raw user input into a format suitable for neural network processing.
  • Text Vectorization: Developed methods to represent text as numerical vectors, such as ASCII-based encoding or custom embeddings, to feed into the neural network.
  • Model Persistence: Implemented model saving and loading mechanisms using PyTorch's torch.save and torch.load for storing trained models and reloading them for inference.
  • Integration with Rasa: Integrated the custom PyTorch model into the Rasa ecosystem, ensuring it processed user input as part of the NLU pipeline and outputted intents with confidence scores.
  • Inference and Prediction: Learned to use the trained model for inference, processing new user messages and predicting intents with confidence scores, enabling dynamic response generation.
  • Evaluation: Gained knowledge on evaluating the model's performance using metrics such as accuracy, precision, recall, and F1-score, while fine-tuning model parameters for better generalization.
  • Rasa Custom Components: Built custom Rasa components (e.g., GraphComponent) to plug the PyTorch intent classifier into the Rasa pipeline, allowing seamless integration into Rasa's architecture.
  • Scalability Considerations: Explored techniques for improving model scalability and handling larger datasets for real-world deployment of a chatbot.
  • Deployment Preparation: Learned how to prepare the model for deployment in a production environment by ensuring efficient model loading and processing for real-time interactions.

def build_model(self, num_intents: int):
    # Neural network with dropout and batch normalization
    return nn.Sequential(
        nn.Linear(100, 50),
        nn.BatchNorm1d(50),
        nn.ReLU(),
        nn.Dropout(0.5),
        nn.Linear(50, num_intents),
    )

def text_to_tensor(self, text: Text):
    # Convert text into fixed-size tensor
    text = text.strip().lower()
    max_length = 100
    vector = [ord(char) % 100 for char in text[:max_length]]  # ASCII-based encoding
    vector += [0] * (max_length - len(vector))  # Padding
    return vector
