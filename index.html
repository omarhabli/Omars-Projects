Graphing Calculator with Custom RISC-V CPU

Project Overview:

  • YouTube demo: https://www.youtube.com/watch?v=B1m-cUdcgSg
  • Final Presentation: https://docs.google.com/presentation/d/10imhudLxiSTwt-bNoYZ7AmsZal4b7ZP1pOPk0xVUYAw/edit?usp=sharing
  • Created RTL block diagrams for a RISC-V single-cycle CPU, QSPI peripherals, and MMIO interface.
  • Used memory-mapped I/O (MMIO) to communicate with a request unit, keypad, screen, and RAM (via Wishbone bus).
  • Integrated QSPI interfaces for real-time output with the user-controlled LCD screen.
  • Built multiply extensions into the CPU’s instruction set based on RISC-V standards.
  • Deployed the full system on silicon using the Caravel SoC for ASIC fabrication as part of Purdue’s NEBULA III initiative.
  • Project GitHub. Source code and test benches within verilog/rtl/team_projects/team_04/src and verilog/dv/team_04/module_tests: nebula-iii-team-04
